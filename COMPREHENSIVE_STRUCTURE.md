# Comprehensive Quiz Database Structure

## Overview

The Lore Master quiz system has been expanded to support a massive content database with:

- **20 Major Topics** covering all popular interests
- **200 SubTopics** (10 per topic) for specific content areas
- **2,000 Levels** (10 per subtopic) with progressive difficulty
- **40,000 Questions** (200 per subtopic, 20 per level)

## Difficulty Progression System

Each subtopic has **10 levels** with progressive difficulty:

### 🌱 Levels 1-2: Beginner
- **Easy questions** about basic facts
- Character names, release dates, obvious details
- Perfect for newcomers to the topic
- Example: "Who is the main character in Naruto?"

### 🌿 Levels 3-4: Intermediate  
- **Medium questions** requiring some knowledge
- Plot details, basic analysis, common facts
- For casual fans with some exposure
- Example: "What village is Naruto from?"

### 🌳 Levels 5-6: Advanced
- **Medium-Hard questions** for dedicated fans
- Deeper plot understanding, character relationships
- Requires more than surface-level knowledge
- Example: "What technique did Minato use to defeat the Nine-Tails?"

### 🏔️ Levels 7-8: Expert
- **Hard questions** for true enthusiasts
- Obscure details, production trivia, fan theories
- Tests deep commitment to the subject
- Example: "What was Kishimoto's original name for Naruto?"

### 🚀 Levels 9-10: Master
- **Very Hard questions** for ultimate fans
- Easter eggs, creator interviews, behind-scenes
- Only the most dedicated fans will know
- Example: "Which panel layout technique did Kishimoto pioneer?"

## Technical Implementation

### Database Structure
```
├── Topics (20)
│   ├── SubTopics (200 total, 10 per topic)
│   │   ├── Levels (2,000 total, 10 per subtopic)
│   │   │   └── Questions (40,000 total, 20 per level)
```

### Quiz Mechanics
- **10 questions per quiz** (randomly selected from level's 20 questions)
- **Progressive unlocking** (must pass level to advance)
- **70% passing score** required to unlock next level
- **Difficulty-based scoring** (harder levels = more points)

### API Endpoints
- `GET /topics` - Browse all 20 topics
- `GET /topics/:id/subtopics` - Explore 10 subtopics per topic
- `GET /subtopics/:id/levels` - View 10 difficulty levels
- `GET /levels/:id/questions` - Get quiz questions

This comprehensive structure provides:
- **Massive Content Variety**: 20 different interest areas
- **Progressive Challenge**: From beginner to master level
- **Scalable Architecture**: Easy to add more content
- **Engaging Gameplay**: Always something new to discover

Perfect for a quiz app that can entertain users for months while they explore their favorite topics and challenge themselves across difficulty levels! 🎯
